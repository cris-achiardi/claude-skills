# Icebreaker Questions for Crazy 8's

These questions help activate divergent thinking before the main ideation session. Select 2-3 questions that fit the problem context. Questions should be provocative, fast to answer (30 seconds each), and push beyond obvious solutions.

## Reframing Questions

**User Perspective Shifts:**
- "What if this was designed for a 5-year-old?"
- "What if this was designed for your grandmother?"
- "What if this was designed for someone with zero tech experience?"
- "What if your most impatient user had to use this?"

**Emotional/Tonal Shifts:**
- "What would the luxury version look like?"
- "What would the playful/fun version look like?"
- "What would the most calming version feel like?"
- "What would the most energetic version feel like?"

## Constraint Removal

**Remove Limitations:**
- "If you had unlimited budget, what would you build?"
- "If technology wasn't a constraint, what's possible?"
- "If you could use magic, what would users experience?"
- "If users had superpowers, what would they do?"

**Time Constraints:**
- "What if users had only 10 seconds to accomplish their goal?"
- "What if users had all day and no rush?"

## Extreme Users

**Power Users:**
- "How would a power user want this to work?"
- "What would an expert need to see immediately?"
- "What shortcuts would professionals demand?"

**First-Time Users:**
- "How would a first-time user need this explained?"
- "What's the most obvious way this should work?"
- "How would someone discover this by accident?"

**Context Extremes:**
- "How would someone in a rush use this?"
- "How would someone use this while distracted?"
- "How would someone use this in the dark?"
- "How would someone use this with one hand?"

## Analogies & Metaphors

**Product Analogies:**
- "What if this worked like Instagram?"
- "What if this worked like a game?"
- "What if this worked like a conversation?"
- "What if this felt like shopping?"
- "What if this was as simple as using a TV remote?"

**Physical Metaphors:**
- "What if this was a physical object on your desk?"
- "What if this was a room you walked into?"
- "What if this was organized like your closet?"

**Interaction Metaphors:**
- "What if users could gesture to control this?"
- "What if users could just speak to it?"
- "What if this felt like flipping through a magazine?"

## Provocations

**Deliberately Opposite:**
- "What's the most boring way to solve this?"
- "What's the most complicated way this could work?"
- "What would the worst possible design look like?"
- "What would make users absolutely hate this?"

**Deliberately Extreme:**
- "What's the most delightful way to solve this?"
- "What would make users smile?"
- "What would be surprisingly beautiful?"
- "What would be unexpectedly fast?"

**Failure Exploration:**
- "What would fail spectacularly but be interesting?"
- "What wild idea probably won't work but might spark something?"
- "What's too ambitious but worth imagining?"

## Sensory Questions

**Visual:**
- "What's the first thing users should see?"
- "What color comes to mind for this?"
- "Should this feel open or focused?"

**Sound:**
- "What's the most calming sound your app could make?"
- "What would the audio version of this be?"
- "Should this be silent or have feedback?"

**Touch/Feel:**
- "How should this feel to interact with?"
- "Should this feel lightweight or substantial?"
- "Fast and responsive or deliberate and thoughtful?"

## Context-Specific Patterns

**For Mobile Apps:**
- "What if this lived on your home screen?"
- "What gesture feels most natural for this?"
- "One tap or multiple steps?"

**For Dashboards:**
- "What's the one metric that matters most?"
- "Detailed or at-a-glance?"
- "What catches your eye first?"

**For Forms/Data Entry:**
- "How would you fill this out while walking?"
- "What's the minimum information you actually need?"
- "Conversation or form?"

## Selection Strategy

**Choose questions that:**
1. Match the problem domain
2. Push against expected solutions
3. Are fast to answer (30 seconds each)
4. Create concrete images in the mind
5. Feel provocative but not overwhelming

**Avoid:**
- Questions that need research to answer
- Questions that are too similar to each other
- Questions that feel like homework
- More than 3 questions total (keep it quick)
